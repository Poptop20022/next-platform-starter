# Страница диагностики подключения

## Как использовать

1. Откройте: `https://your-site.netlify.app/test-connection`
2. Введите URL вашего backend
3. Нажмите "Запустить тесты"
4. Просмотрите результаты

## Что проверяется

- ✅ Формат URL
- ✅ Доступность backend (health check)
- ✅ Существование auth endpoint
- ✅ Настройки CORS
- ✅ Переменные окружения
- ✅ Сетевая доступность
- ✅ Основные маршруты API

## Интерпретация результатов

### ✅ Success (зеленый)
Все работает правильно

### ❌ Error (красный)
Критическая проблема, требует исправления

### ⚠️ Warning (желтый)
Потенциальная проблема, стоит проверить

### ℹ️ Info (синий)
Информационное сообщение

## Частые проблемы

### 404 на /api/auth/login
- Проверьте, что backend запущен
- Проверьте маршруты в backend/src/routes/index.ts
- Убедитесь, что роуты правильно экспортированы

### CORS ошибки
- Проверьте настройки CORS в backend/src/index.ts
- Убедитесь, что ваш Netlify домен добавлен в allowedOrigins

### Network error
- Проверьте, что backend URL правильный
- Проверьте, что backend доступен из браузера
- Проверьте firewall настройки
